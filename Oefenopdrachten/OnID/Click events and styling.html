Opdracht uitleg

In deze oefening gebruikt je 6 div elementen van 200 x 200 pixels. Deze DIV objecten hebben een zwarte solid border van 1 pixel en staan als display:inline-block.
Geef ieder vierkant via het onclick event (dus wanneer je op het element klikt) via DOM functies de volgende eigenschappen:

  • vierkant 1: een rode achtergrondkleur
  • vierkant 2: een border van 2 pixels
  • vierkant 3: een rotatie van 45 graden
  • vierkant 4: verklein het object met de helft
  • vierkant 5: maak onzichtbaar
  • vierkant 6: start de animatie 'roteer' op dit object
  • vierkant 7: stel een achtergrond afbeelding (no-repeat center) in op: https://www.rocmn.nl/themes/custom/rocmn/logo.svg?v=1538394935

Programmeer de reset-knop zodat je alle vierkanten weer terugzet naar hun standaard waarden.

Tip:Gebruik voor deze laatste opdracht de eigen functie "reset" welke je aanroept via het onclick event van de knop...
